# html-insights — rendered HTML reports

Central collection of self-contained, browser-openable HTML reports produced by the
`docs/dev/` investigations. Each file base64-embeds its own figures, so it is fully
self-contained — download or open it directly, no sibling assets needed.

These are **rendered outputs**, not the source of truth. Each is regenerated by a
generator that lives with its workstream; re-run the generator to refresh the copy here
(the generator writes both its workstream copy and the mirror in this folder).

| file | generated by | workstream |
|---|---|---|
| `transition_trigger_investigation.html` | `docs/dev/transition/cleanroom/make_transition_report.py` | implicit→momentum transition trigger (clean-room): why the cooling-balance trigger stalls under hybr; the transition is geometric, not thermal |

To add a future report: have its generator also write here (mirror its `OUT` to
`docs/dev/html-insights/<name>.html`) and add a row above.
